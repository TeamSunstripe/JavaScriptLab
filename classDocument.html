    new Date()

    STYLE

    DISPLAY

    秒数を指定

    date = new Date(msec);
    ＊ msecは秒数が入ってます

    フォーマットに対応した文字列を指定

    date = new Date("Thu, 06 Sep 2017 00:00:00 +0900");

    日付の文字列を指定

    date = new Date("2017/11/12 10:00:00");

    Date.getYear


    Date.prototype.getYear は 1900 年からの年数で非推奨。

    STYLE

    DISPLAY

    日付データの年を表示


    ＊ date = new Date("Thu, 06 Sep 2017 12:00:00 +0900")を指定 date.getYear();

    Date.getFullYear


    Date.prototype.getFullYear を使う。

    STYLE

    DISPLAY

    日付データの年を表示

    ＊ date = new Date("Thu, 06 Sep 2017 12:00:00 +0900")を指定
    date.getFullYear();

    Date.getMonth


    getMonth を使う。
    new Date の第 2 引数でも、Date.prototype.getMonth でも、月は 0-11で表す。

    STYLE

    DISPLAY

    日付データの月を表示

    ＊ date = new Date(2017, 9, 19);を指定
    date.getMonth();


    Date.getDate


    getDate を使う。
    new Date の第 2 引数でも、Date.prototype.getDate でも、日付は 1-31で表す。

    STYLE

    DISPLAY

    日付データの 日付を表示

    ＊ date = new Date(2017, 9, 19);を指定
    date.getDate();


    Date.getDay


    Date.prototype.getDay は 0 を日曜とする曜日を返す。

    STYLE

    DISPLAY

    日付データの 曜日（数字）を表示

    ＊ date = new Date(2017, 9, 19);を指定
    date.getDay();

    曜日（数字）を月ー日で指定する場合

    例：
    daysData = [ {title:"日"}, {title:"月"}, {title:"火"}, {title:"水"}, {title:"木"}, {title:"金"}, {title:"土"}, ];
    daysData[date.getDay()].title
